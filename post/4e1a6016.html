山东大学 计算机科学与技术 学院

云计算技术 课程实验报告

学号：202100130017 姓名：蒋全盛 班级：计科3班
实验题目：利用主流云平台搭建个人博客或网站
实验学时：2 实验日期： 2024/3/22

实验目的：熟悉使用主流云平台并搭建个人博客或者网站。

硬件环境：联网的计算机一台

软件环境：Windows or Linux

实验步骤与内容

1.注册 github

2.安装 Node.js

3.连接 Github 连接 SSH 密钥

4.创建 Github Pages 仓库

5.本地安装hexo

6.初始化Hexo博客

7.挂载到github

8.部署项目到Github


9.博客网址：曦の博客 (xizhiwai.github.io)

结论分析与体会

  1. 此次实验通过Node.js、Hexo以及Github页面实现了个人网站的搭建，使我对主流云平台的使用有了深刻的理解和掌握。
  2. 使用Nodejs和Hexo进行博客搭建，大大提升了编写博客的便利性和效率。Hexo提供了许多强大的功能和主题，使个人博客具有了更好的用户体验和视觉效果。
  3. 实验过程中遇到了一些难题，例如Hexo的配置、博客主题的选择等，但通过查阅相关资料以及不断尝试，问题被逐一解决，增强了我解决问题的能力，也增加了我对工程实践的信心。
  4. 搭建个人网站是一个相当有意义的项目，网站不仅仅是一个展示自己的平台，也是记录自己学习和生活的地方，令人收获颇丰。
  5. 学会使用云平台和Hexo等工具后，我可以在任何地方、任何时候对我的网站进行更新和维护，这给我的学习和工作带来了极大的便利。
  6. 当我看到自己搭建的个人网站在网络上成功运行，很开心，同时也更加坚定了我继续学习和掌握新知识、新技术的决心。
